## 수행 작업

### 1. 수익의 3대 축 점검

#### 트래픽 볼륨 (SEO)
- 모든 페이지에 고유 title/description 설정 여부
- sitemap.xml 정상 동작 여부
- 구조화 데이터 (JSON-LD) 적용 여부
- 내부 링크 구조 적절 여부
- 도구 설명 콘텐츠 충분 여부 (300자 이상)

#### 페이지 속도 (Core Web Vitals)
- LCP, INP, CLS 목표치 달성 여부
- 번들 크기 예산 준수 여부
- 이미지/폰트 최적화 상태

#### 광고 가시성 (Viewability)
- 도구 결과 하단 InContentAd 배치 여부 (가장 효과적)
- SidebarAd sticky 설정 (데스크톱)
- below-fold 광고 lazy loading 적용 여부
- 광고 슬롯 고정 치수 설정 여부

### 2. 에드센스 정책 준수 검사
- 콘텐츠:광고 비율 70:30 이상
- 에러/404 페이지에 광고 없음
- 광고 근처 오클릭 유도 요소 없음
- 모바일 above-the-fold 광고 최대 1개
- 광고 라벨 올바르게 표시
- 도구 버튼-광고 간 최소 32px 간격

### 3. 광고 배치 효과 분석
- 각 페이지별 광고 수 집계
- 광고 유형별 배치 현황 (Display, Multiplex, In-article)
- 모바일 vs 데스크톱 배치 차이 확인

### 4. 개선 기회 식별
- 광고가 없는 페이지 식별
- 최적화되지 않은 배치 감지
- 새 도구/페이지 추가로 수익 증가 가능한 영역
